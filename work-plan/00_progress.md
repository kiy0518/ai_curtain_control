## 프로젝트 개요

**컨셉**: 인공지능 카메라로 손 제스처를 인식해 커튼을 열고/닫고, 사용자별 맞춤 스케줄로
자동 제어하는 **스마트 AI 커튼**. 대시보드(웹/모바일)와 원격 제어를 지원.

**핵심 흐름**:
```
IMX415 카메라 → RK3576 NPU(손 키포인트) → 제스처 인식 → 커튼 모터 제어
                                              │
                          웹/모바일 대시보드 ◄┘ (스트리밍·설정·스케줄)
                                              │
                                  원격 접속(고유 주소)
```
