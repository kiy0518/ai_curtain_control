# 마스터 플랜 — Smart AI Curtain

> 작성일: 2026-06-10

---

## 1. 비전

> "카메라 앞에서 손짓 한 번으로 커튼을 여닫고, 내 생활 패턴을 학습해 알아서 움직이는 커튼."

- **엣지 우선**: 영상/추론은 보드(RK3576 NPU)에서 로컬 처리 (프라이버시·저지연).
- **무접점 제어**: 리모컨/스위치 없이 손 제스처 + 스케줄.
- **개인 맞춤**: 사용자별 스케줄·선호 개폐량, (선택)사용자 식별.
- **어디서나**: 웹/모바일 대시보드 + 고유 주소로 원격 제어.

---

## 2. 시스템 아키텍처 (목표)

```
┌───────────────── ROCK 4D (엣지 디바이스) ─────────────────┐
│ IMX415 ─► ISP 캡처 ─► RKNN NPU(손키포인트) ─► 제스처 엔진   │
│                                   │                        │
│   모터 컨트롤러(시리얼) ◄─ 제어로직 ◄┼◄── 스케줄러/자동화     │
│   (UART, 개/폐/정지·속도고정)      │◄── BLE 서버(GATT) ◄────┼─BLE─ Flutter 리모컨 앱
│                                   ▼                        │
│         앱 서버(stdlib) ── REST/MJPEG ── 대시보드(PWA)       │
└───────────────────────────────────┼────────────────────────┘
                                     ▼  (고유 주소/터널)
                          웹/모바일(PWA) · 원격 사용자
```
> 제어 입력 = 제스처 / 대시보드(웹) / 스케줄 / **BLE 리모컨 앱** — 모두 단일 `CurtainController` 로 수렴.

핵심 모듈(목표):
- **vision/**: 캡처·추론·제스처 (현재 `src/`).
- **control/**: 커튼 상태머신(열림/닫힘/정지/부분%) + 모터 드라이버 추상화.
- **app/**: API 서버 + 대시보드(웹/PWA) + 인증.
- **scheduler/**: 시간·일출/일몰·조도·재실 기반 자동화.
- **remote/**: 등록기 + 고유 주소(터널/게이트웨이).

## 4. 버전 관리 정책

- **SemVer 기반**: `vMAJOR.MINOR.PATCH`. 1.0 이전(`v0.x`)은 개발 단계.
- **MINOR**: Phase 1개 완료 시 +1 (예: Phase1 완료 → v0.2.0).
- **PATCH**: 버그픽스/소규모 개선.
- **태그 + 릴리스**: 각 버전마다 annotated 태그 생성. 태그/릴리스 노트에 반드시 포함:
  1. **생성 시각**(YYYY-MM-DD HH:MM:SS)
  2. **이번 버전 진행 내용**
  3. **미해결 / 다음 버전 승계 항목**
- **브랜치**: `main`(안정) + 기능 개발은 `feat/<phase>-<name>` 브랜치 → PR/머지(1인 개발 시 직접 머지 가능).
- **진행상태**: `00_progress.md` 의 Phase 표 + 각 문서 체크박스를 작업 시 갱신.

예) 태그 생성:
```bash
git tag -a v0.2.0 -m "v0.2.0 ...(시각/진행/승계)..."
git push origin v0.2.0
```

## 5. 기술 스택 (잠정)

| 영역 | 선택(안) |
|------|---------|
| 엣지 추론 | rknnlite (RK3576 NPU), OpenCV |
| 앱 서버 | FastAPI + uvicorn (WebSocket 지원) |
| 프론트 | 경량 HTML/JS → PWA (필요시 Vue/React) |
| 모터 제어 | 외부 모터 컨트롤러 + **시리얼(pyserial, UART)** · 속도 고정 |
| 스케줄 | APScheduler, astral(일출/일몰) |
| 원격 | 등록기 + FRP/cloudflared (Azone_Gateway 패턴 참고) |
| 저장 | SQLite (설정·스케줄·프로파일·이력) — 스키마는 `04_*` 데이터모델 절 |
| 리모컨 앱 | **Flutter**(Material 3) + **BLE**(`flutter_blue_plus`) |
| BLE 서버(보드) | BlueZ + Python(`bless`/`bluezero`) GATT 주변장치 |
| 음성(STT) | 오프라인 — vosk/sherpa-onnx/whisper.cpp + 웨이크워드 (클라우드 X) |
| 온디바이스 LLM | RKLLM(NPU) 소형(Gemma 3n E2B/Qwen2.5-1.5B), 온디맨드 (클라우드 X) |

> **조기 결정 필요**: **단일 계정 vs 다중 계정** — SQLite 스키마·인증(`07_*`)·개인화(`05_*`)에 영향.
