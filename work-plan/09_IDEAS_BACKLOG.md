# 09. 아이디어 · 확장 백로그 (제안)

> 작성일: 2026-06-10 · 정식 Phase 편입 전 아이디어 모음. 우선순위는 협의.

## A. 제스처/상호작용 강화
- **활성화 제스처**: 아무 동작이나 오인식 방지 위해 "손 들고 3초" 등 트리거 후에만 명령 수신. (오작동 ↓, 중요)
- **부분 개폐 직관화**: 손 높이(손목 y좌표)로 개폐량 연속 조절 — "손 올리면 더 열림".
- **양손 제스처**: 두 손 거리로 개폐폭 조절, 한 손은 선택/한 손은 양.
- **방향 스와이프**: 손을 좌/우로 쓸면 열림/닫힘 (직관적, 키포인트 이동벡터로 검출).

## B. 자동화/지능
- **이불 모드/취침 감지**: 늦은 시간 + 어두움 → 자동 닫기.
- **눈부심 방지**: 햇빛이 특정 각도로 강할 때 부분 차광(조도+시간).
- **재실/외출 연동**: 사람 없으면 보안상 닫기, 귀가 시 열기.
- **학습형 스케줄**: 사용 이력으로 개폐 시간 자동 제안/적용.

## C. 멀티 / 확장
- **멀티 커튼·존**: 방마다 디바이스, 대시보드에서 일괄/개별 제어.
- **멀티 카메라**: 큰 창/여러 각도.
- **Home Assistant / MQTT 연동**: 기존 스마트홈에 편입(엔티티 노출).
- **음성 명령 병행**: 경량 wake-word + 명령("커튼 닫아").

## D. 기술/품질
- **WebRTC 스트리밍**: MJPEG보다 저지연·저대역. 모바일 원격에 유리.
- **하드웨어 JPEG 인코딩(mppjpegenc)**: CPU 부하 ↓ (스트리밍).
- **INT8 양자화 + 입력크기 튜닝**: NPU 처리량 최적화.
- **온디바이스 손+얼굴 멀티모델**: 사용자 식별 → 개인화.

## E. 안전/신뢰
- **끼임/장애물 감지**: 모터 전류 급증 또는 영상 변화로 정지(안전 필수).
- **수동 우선/페일세이프**: 정전·오류 시 안전 상태(닫힘 등) + 물리 버튼.
- **감사 로그**: 누가/언제/무엇을(원격 제어 추적).

## F. 제품화
- **모바일 앱(PWA)**: 설치형 경험, 푸시 알림(스케줄 알림).
- **온보딩 마법사**: 커튼 끝단 캘리브레이션, 위치/시간대 설정.
- **에너지 리포트**: 채광/단열 관점 사용 통계(부가가치).
