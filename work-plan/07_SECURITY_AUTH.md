# 07. 인증 · 보안

> Phase 5 · 목표버전 v0.6.0 · 상태: **구현됨**(단일 관리자 로그인)

## ✅ 구현 (src/auth.py + app.py)
- 단일 관리자 **비밀번호 로그인**(PBKDF2-HMAC-SHA256+salt, SQLite 저장).
- **세션 쿠키**(HttpOnly, SameSite=Lax, 7일) — 대시보드/모든 API 보호. 미인증 401, `/`→로그인 페이지.
- 로그인 **시도 제한**(5회 실패→30초 잠금). 비밀번호 변경 시 전 세션 무효화.
- 기본 비밀번호 `admin` → 대시보드 경고 배너 + 변경 UI. 로그아웃.
- 검증: 미인증 401 / 로그인 / 제어 게이트 / 변경·세션무효화 모두 통과.

### 남은 강화 (추후)
- HTTPS는 원격(터널)이 TLS 종단; 로컬 직접접속은 평문(내부망).
- 다중 계정/역할(현재 단일 관리자), 토큰(외부 API), per-IP 레이트리밋.

## 목표
대시보드/제어 API를 인증된 사용자만 접근. 원격 노출 전 필수.
